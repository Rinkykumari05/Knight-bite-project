## 📌 About The Project

Knight Bites is a **responsive food ordering frontend website** designed to simulate a modern restaurant UI.

This project focuses on:
- Clean UI/UX design
- Responsive layouts
- Real-world website structure

---

## ✨ Key Features

- 🍔 Food menu display
- 📱 Fully responsive design
- 🎨 Attractive UI with Bootstrap
- 🔗 Smooth navigation between pages
